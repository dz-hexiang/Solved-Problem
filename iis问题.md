>32位机器：

```
C:\Windows\Microsoft.NET\Framework\v4.0.30319\aspnet_regiis.exe -i
```

>64位机器：

```
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\aspnet_regiis.exe -i
```
