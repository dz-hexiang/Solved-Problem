#### 没有nodejs 的时候执行下面这句 

```
apt-get update
```

#### 安装nodejs

```
sudo apt-get install -y nodejs
```

#### 当安装好nodejs后 node 命令依然无效时候执行

```
sudo apt-get install -y nodejs-legacy
```

#### nodejs 软件管理工具

```
sudo apt-get install -y npm
```

#### 查看版本

```
node --version
```
